Welcome to the Anova Style Guide

This document is an example of CSS styling for the Anova web properties and includes element names, how they are used, and copiable CSS code.

<h1> Heading 1

<h1> is used for titles such as the page title and recipe title.

color: #646464;
font-family: "proxima-nova",sans-serif;
font-style: normal;
font-weight: 300;
}
